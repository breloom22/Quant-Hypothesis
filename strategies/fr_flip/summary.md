# Summary: fr_flip

## 핵심 수치
- **최적 전략**: FR 부호 전환 반대 방향 진입 (예: +→- 시 숏 진입)
- **성과**: PF=1.29, 승률=56.9%, 샤프비율=1.13, MDD=20.3%
- **최적 파라미터**: 8시간 봉, 신호 확인 후 1봉 지연 진입, 4봉(32시간) 보유, 익절 1.5%, 손절 없음
- **워크포워드**: 5-fold 중 4-fold 통과, 평균 PF=1.17
- **종목**: BTC에서만 유효 (알트코인 대부분 실패)

## 주요 결론
1. **메커니즘 반전**: FR 전환은 과밀 해소 완료 신호가 아니라, 이미 진행 중인 반대 방향 추세의 지연된 확인 신호로 작용함.
2. **BTC 고유성**: 신호는 BTC의 시장 전체 레버리지 청산 트리거 역할에서 비롯되며, 알트코인으로 확장 불가.
3. **손절 미적용**: 반전 전략 구조상 진입 후 역방향 노이즈는 정상 경로로, 손절 설정 시 성능 악화됨.
4. **비대칭성**: BTC 상승 편향으로 롱(승률 75%)이 숏(승률 62.7%)보다 우수함.

## 구조적 한계
- BTC 단일 종목 의존성으로 분산 불가
- 무손절 구조로 테일 이벤트에 취약
- 8시간 펀딩비 정산 주기에 전략이 종속됨
- 낮은 신호 빈도(월 2~5회)로 통계적 신뢰도 확보 어려움
